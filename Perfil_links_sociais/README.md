# PROJECT: Perfil de links sociais#

## Sobre: - Esse projeto, tem um layout basico de uma lista de redes sociais de um usuario.

## Ferramentas utilizadas

    - Html(semantica)
    - Css e sass(estilos)
  
## Paleta de cores:

- Green: hsl(75, 94%, 57%)
- White: hsl(0, 0%, 100%)
- Grey 700: hsl(0, 0%, 20%)
- Grey 800: hsl(0, 0%, 12%)
- Grey 900: hsl(0, 0%, 8%)
